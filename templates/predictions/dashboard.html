🤖 How Ensemble Voting Works

This system uses 12 different ML models and combines their predictions using majority voting:

✓ Models Used:
  • Logistic Regression
  • Decision Tree
  • SVM (Linear SVC)
  • K-Nearest Neighbors
  • Naive Bayes
  • Voting Classifier
  • Random Forest
  • AdaBoost
  • Gradient Boosting
  • LightGBM
  • XGBoost
  • Probabilistic LR
🗳️ Voting Process:
  1. Each model makes a prediction
  2. Count votes for each crop
  3. Select crop with most votes
  4. Calculate confidence as % of votes
  5. Display with crop emoji/image
📋 Sample CSV Format

Your CSV file should look like this:

N,P,K,temperature,humidity,ph,rainfall
40,53,34,25.5,82,6.5,220
60,45,28,28,75,7,180
20,30,25,22,88,5.5,300

ℹ️ Column names are case-sensitive. Ensure exact spelling: N, P, K, temperature, humidity, ph, rainfall

📚 About Parameters
Soil Parameters:
  • N (Nitrogen): 0-150 mg/kg - Essential for leaf growth
  • P (Phosphorus): 0-150 mg/kg - For root and flowering
  • K (Potassium): 0-150 mg/kg - For overall plant health
Climate Parameters:
  • Temperature: 0-60°C - Average growing season temp
  • Humidity: 0-100% - Relative humidity
  • pH: 3.5-9.5 - Soil acidity/alkalinity
  • Rainfall: 0-350mm - Annual precipitation
